# 🏐 VolleyHub

VolleyHub is a modern web application focused on volleyball leagues.
It provides information about teams, matches, standings and basic knowledge about the sport.

---

## 📌 About This Project

This project is the **first of two assignments** for the course:

**"Σχεδίαση Εφαρμογών & Υπηρεσιών Διαδικτύου"**

It serves as the **base implementation**, which will be further expanded and completed in the second project.

---

## 📌 Features

* 📋 Team management (add new teams)
* 🏐 Match display system
* 📊 League table
* 🎥 Local video highlights for each team
* 🖼️ Media-rich content (images & logos)
* 🤝 Sponsors section with external links
* 📱 Fully responsive design (3-tier: mobile, tablet, desktop)

---

## 📱 Responsive Design

The application follows a **3-tier responsive approach**:

* 📱 Mobile (≤768px)
* 📲 Tablet (768px – 1024px)
* 💻 Desktop (>1024px)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* Responsive Design (Media Queries)
